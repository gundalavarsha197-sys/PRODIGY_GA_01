# 🧠 AI Smart Text Generator

An AI-powered web application built using Streamlit and Hugging Face Transformers that generates intelligent text based on user input.

🚀 Features:
- Generate text using AI (GPT-2 model)
- Customize output length
- Generate multiple results
- Fast response with caching
- Download generated text
- Simple and user-friendly interface

🛠️ Tech Stack:
- Python
- Streamlit
- Transformers (Hugging Face)
- Torch
- Torchvision

📂 Project Structure:
AI_Text_Generator/
│── app.py
│── requirements.txt
│── README.md

▶️ How to Run:
1. Clone the repository:
   git clone https://github.com/your-username/AI_Text_Generator.git
   cd AI_Text_Generator

2. Create virtual environment:
   python -m venv myenv
   myenv\Scripts\activate

3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   streamlit run app.py

💡 Usage:
- Enter a prompt in the text box
- Click "Generate 🚀"
- View AI-generated results
- Download output if needed

📌 Example Prompts:
- Write a short story about sustainable farming
- Explain artificial intelligence in simple words
- Generate a motivational quote for students

⚠️ Note:
- Use Python 3.10 or 3.11 for best compatibility
- First run may take time due to model download
